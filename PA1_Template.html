Conclusions:

The shape of each histogram is the same, approximately normal, and while the means are the same (using the interval replacement method) the medians differ slightly (10765 on the original data set versus 10766.19 on the data set with synthesized values). This should be expected as the number of values is increased on the second data set but slightly and therefore the median should also change (assuming the NA values are ignored in the original calculation of the median on the original data set). It should also be noted that on the sythesized data set the number of days (the frequency) is slightly higher with the max frequency reaching approximately 35 while on the original dataset the maximum frequency was approximately 25. This would make sense as the substitute values should increase the number of days with valid values thereby increasing the height of the some values in the histogram.
